# qt_img_compress 

#### 介绍
这是一个在线图片压缩的工具，支持批量处理，但文件处理，覆盖源文件等。
支持jpg，jpeg，png格式的文件处理。

#### 软件架构
软件使用的Qt版本为5.11


#### 安装教程
x86-64下载发布的deb包进行安装，其他架构需自行编译安装

#### 参与贡献
感谢resmush网站提供图片处理接口，https://resmush.it/api
