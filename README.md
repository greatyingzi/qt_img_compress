# qt_img_compress 

#### 介绍
这是一个在线图片压缩的工具，初始版本每次只支持处理一张图片。

#### 软件架构
软件使用的Qt版本为5.15


#### 安装教程
目前只支持自行编译安装

#### 参与贡献
感谢resmush网站提供图片处理接口，https://resmush.it/api
